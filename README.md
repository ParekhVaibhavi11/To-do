# To-Do List Application

A dynamic web application for efficient task management, built with EJS, Node.js, Express.js, and MongoDB.

## 🚀 Live Demo

* **Live URL:** [https://to-do-list-project.onrender.com/](https://to-do-list-project-zvod.onrender.com/)

## ✨ Features

* **Task Creation:** Add new tasks with a title and a priority level.
* **Task Editing:** Update task details, including priority, using a user-friendly edit form.
* **Task Deletion:** Permanently remove tasks to keep your list organized.
* **Data Persistence:** All task data is stored securely in a MongoDB Atlas database.
* **Dynamic UI:** The interface updates in real-time as tasks are added, edited, or deleted.
* **Form Validation:** Prevents empty tasks from being created or updated.

## 🛠️ Technology Stack

* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Database ODM:** Mongoose
* **Templating Engine:** EJS
* **Environment Variables:** Dotenv
* **Frontend:** HTML5, CSS3, JavaScript

## 📦 Deployment

This application is deployed on Render. The process is straightforward:

1.  Push your code to a GitHub repository.
2.  Create a new Web Service on Render and connect your GitHub repo.
3.  Set the `MONGODB_URI` environment variable with your MongoDB Atlas connection string.
4.  Set the **Build Command** to `npm install` and the **Start Command** to `node index.js`.
5.  Render will handle the rest, providing you with a live URL.
